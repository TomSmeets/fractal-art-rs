# Fractal art (Rust)

A simplified rust implementation of [FractalArt](https://github.com/TomSmeets/FractalArt).

## Usage

```
USAGE:
    fractal-art [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -c, --center <center>
    -o, --output <output>
        --seed <seed>
    -s, --size <size>
```
